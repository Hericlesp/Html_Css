ENTENDA A HTML

O HTML é a base da web. conheça sua historia, curiosidade e aplicações

O que é HTML??

HTML significa **HyperText Markup Language** e é a linguagem padrão para criar páginas web. Ele foi criado por Tim Berners-Lee em 1991 como uma maneira de compartilhar informações de forma estruturada na internet.

HTML é uma linguagem de marcação que utiliza **tags** para definir a estrutura do conteúdo de uma página. Ele é usado para inserir texto, imagens, vídeos, links e outros elementos visuais em sites.

ESTRUTURA BASICA!!

Toda página HTML começa com a declaração <DOCTYPE html> para informar ao navegador que o documento segue o padrão HTML5. A estrutura básica inclui: 


                    
                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title>Título da Página</title>
                            </head>
                            <body>
                                <h1>Bem-vindo ao HTML!</h1>
                            </body>
                        </html>

PRINCIPAIS TAGS HTML

  • <h1> a <h6>: Títulos e subtítulos
  • <p>: Parágrafos
  • <img>: Imagens
  • <a>: Links
  • <div>: Divisões ou contêineres

POR QUE APRENDER HTML??

Aprender HTML é o primeiro passo para criar sites e aplicações web. Ele é fundamental para qualquer desenvolvedor, seja para projetos simples ou complexos.

Ao entender HTML, você pode personalizar a aparência do conteúdo com CSS e adicionar funcionalidades com JavaScript.

Tabela em HTML

As tabelas em HTML são estruturas utilizadas para organizar e apresentar dados em formato bidimensional, com linhas e colunas. Criadas com o elemento <table>, elas são amplamente usadas para exibir informações tabulares, como relatórios, horários, comparativos ou qualquer tipo de dado que precise de uma disposição ordenada. Dentro de uma tabela, outros elementos complementares ajudam a estruturar as informações:

Tags de Tabela em HTML
Tag/Atributo Descrição
<tr> Elemento de linha de tabela. Usado para adicionar linhas a uma tabela antes de adicionar dados e títulos da tabela.
<td> Elemento de dados da tabela. Pode ser aninhado dentro de um elemento de linha da tabela (<tr>) para adicionar uma célula de dados.
<thead> Elemento de cabeçalho da tabela. Define os títulos das colunas da tabela encapsulados em linhas (<tr>).
rowspan (Atributo) Indica quantas linhas uma célula deve abranger. Exemplo: 
<td rowspan="2">Célula grande</td>
<tbody> Elemento do corpo da tabela. Contém todos os dados da tabela, exceto o conteúdo de cabeçalho e rodapé.
<th> Elemento de cabeçalho da tabela. Usado para adicionar títulos às linhas e colunas.
colspan (Atributo) Indica quantas colunas uma célula deve abranger. Exemplo: 
<th colspan="2">Cabeçalho grande</th>
<tfoot> Elemento de rodapé da tabela. Usado para fornecer conteúdo de rodapé ou resumir o conteúdo no final de uma tabela.
<table> Elemento de tabela em HTML. Representa uma tabela bidimensional composta de linhas e colunas.
